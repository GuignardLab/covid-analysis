# Plotting some data with this notebook

To run the notebooks you can either clone/download this repository and use jupyter notebook or run it via google colab [there](https://colab.research.google.com/github/GuignardLab/covid-analysis/blob/main/Data-analysis-Colab.ipynb).

If you want to run it on your own computer you will need the following libraries:
- numpy
- scipy
- matplotlib
- pandas

you can install them using anaconda/miniconda and the following command:

```shell
conda install numpy scipy matplotlib pandas
```

or with pip:
```shell
pip install numpy scipy matplotlib pandas
```
